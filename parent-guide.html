画面時間と交換

「もう少し遊びたい」と言われても、「ポイントが足りないね、タスクをやるか勉強するかしよう」と返すだけ。 親は判定するだけ、子どもは自分で計画する。怒鳴る・ねだる・ケンカする時間が消えます。

親は怒らない、説得しない、罪悪感もない。

「いいよ、ポイント使う?15分=50ptだよ」

3

「残り20pt、もう厳しいね。明日もある。」
➜ 子ども: しぶしぶ納得。「ゲーム→今日はもうムリ→寝る」が自然な流れに。
ルールは数字。親は悪者じゃなくなる。
4
「学校から帰ってきた、何しよう?」
— 何も言う必要なし —
➜ 子ども: アプリを開く →「今日のタスクが3つ残ってる、あと60pt稼げる」→ 自分で進める。
『指示待ち』が『自分で考える』に変わる。

🧠 なぜこれが効くのか

🛡️
親が悪者じゃなくなる
ルールを決めているのはアプリ。「ダメと言うパパ・ママ」から「一緒に応援する人」へ役割が変わります。
🎯
子どもに主導権が戻る
「やらされる」じゃなく「自分のために稼ぐ」。内発的動機(自分でやりたい気持ち)が育ちます。
💰
お金もお菓子もいらない
ご褒美が「自分の好きな時間」だから、物質的なご褒美に頼らずに動機づけができます。
📚
勉強が苦痛じゃなくなる
「勉強しなさい」じゃなく「ポイント稼ぎたいから自分でやる」。学習5分で10pt、塵も積もって30分の追加ゲーム時間に。
セットアップ

5分でスタート

最初の1日で仕組みを作れば、あとは子どもが自分で回します。

1
アプリをダウンロードして親アカウントを作る
App Storeから無料ダウンロード。「保護者」を選んでメール登録。続けてお子さまの名前と学年を入力すると、お子さま専用のログイン情報が作成されます。
2
ご褒美 = 画面時間を設定する
お子さまがウィッシュリストから自分でリクエストできます。親は承認するだけ。最初は親側で例を入れておくとスムーズです。
設定例 ・ゲーム30分 = 100pt
・YouTube15分 = 50pt
・iPad自由時間1時間 = 200pt
・友達と外出1時間 = 80pt
3
ポイントの稼ぎ方 = タスクを設定する
毎日のタスク・お手伝い・宿題・勉強など、ポイント源となる行動を登録します。タスク完了 → 親が承認 → ポイント加算の流れ。
設定例 ・歯みがき = +10pt (毎日)
・宿題 = +30pt (毎日)
・お皿洗い = +15pt (週2回)
・部屋の掃除 = +50pt (週末)
・読書30分 = +20pt (毎日)
4
学習で追加ポイント — 勉強もご褒美に
アプリ内に小1〜小6の算数・国語・理科・社会のクイズを約1万問収録。1問正解で2pt、5問続けて全問正解でボーナス10ptなど。子どもは「あと20pt欲しい」と思ったら、自分で勉強しに行きます。
学習で稼げる例 ・5問正解 = +10pt
・10問連続正解 = +25pt (ボーナス)
・毎日続けると「ストリーク」が伸びてさらにボーナス
5
完成。あとは見守るだけ。
お子さまは自分のホーム画面で「今日のタスク」「ポイント残高」「次のご褒美まであと何pt」を見ながら、自分で計画します。
親は提出があったとき承認するだけ。「やりなさい」のリマインダーが激減します。
そのほかの機能

続けやすくする仕組み

「最初の3日は燃えても、1週間で挫折」を防ぐためのデザイン。

🐠
バーチャルペット
タスクと勉強でペットが進化。9種類の冒険コース(海・森・宇宙・恐竜など)。子どもにとって「続ける理由」になります。
🔥
連続日数(ストリーク)
毎日アプリを開くと記録が伸びます。3日・7日・14日でバッジを獲得。習慣化を強くサポート。
📊
毎週レポート
お子さまの完了率・科目別の成長を毎週まとめて確認。「今週は算数が伸びた」がひと目でわかります。
👨‍👩‍👧
パパも一緒に管理
共同保護者を招待して、家族で同じダッシュボードを共有。誰が承認してもOK。
🌙
休みの日設定
日曜日や旅行中は「ペナルティなし・ストリーク継続」に。家族の予定に合わせられます。
🔔
通知も自動でやさしく
21:00〜7:00は通知サイレント。お子さまにストレスを与えない設計。
🛡️
プライバシーを守る設計
COPPA準拠。広告なし・チャット機能なし・第三者へのデータ販売なし。 すべて家族内で完結し、お子さまの情報は最小限のみ保存されます。
